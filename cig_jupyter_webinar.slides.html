What is Jupyter?¶

Project Jupyter is a collection of tools for interactive and exploratory data analysis, including tools for sharing that analysis.

    feature['properties']['style'] = {'color': 'grey','weight': 1,'fillColor': 'grey','fillOpacity': 0.5}
